# Assets Directory

  This directory should contain:

  ## Required Images
  - `icon.png` - App icon (1024x1024)
  - `splash.png` - Splash screen (1284x2778)
  - `adaptive-icon.png` - Android adaptive icon (1024x1024)
  - `favicon.png` - Web favicon (48x48)

  ## Design Guidelines
  - Use Islamic green (#047857) as primary color
  - Include gold accents (#D97706)
  - Consider Islamic geometric patterns
  - Incorporate Kaaba or mosque imagery

  ## Temporary Solution
  The app will work without these images, but you should add them for a professional look.

  You can use online tools like:
  - Canva: https://www.canva.com
  - Figma: https://www.figma.com
  - Adobe Express: https://www.adobe.com/express

  ## Quick Asset Generation
  For testing, you can:
  1. Create simple colored squares in the right dimensions
  2. Use the company name "AL BURHAN" as text
  3. Add icons from https://icons8.com or https://flaticon.com
