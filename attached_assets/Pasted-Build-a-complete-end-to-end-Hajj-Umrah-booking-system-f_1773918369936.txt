1️⃣ CUSTOMER LOGIN SYSTEM

• Login via OTP (SMS + WhatsApp)
• Fields:
  - Name
  - Mobile
  - Email

• After login → redirect to dashboard

-----------------------------------------------------

2️⃣ PACKAGE SELECTION

Customer can:

• View Packages:
  - Hajj
  - Umrah

• Click "Book Now"

-----------------------------------------------------

3️⃣ BOOKING REQUEST SYSTEM

Customer submits request:

Required fields:

• Name
• Mobile
• Passport Number
• Date of Birth
• Passport Issue Date
• Passport Expiry Date
• Place of Issue

👉 Save as: PENDING

-----------------------------------------------------

4️⃣ ADMIN DASHBOARD

Admin can:

• View booking requests
• Approve / Reject

• Status = APPROVED
• Enable payment for customer

-----------------------------------------------------

5️⃣ PAYMENT SYSTEM (RAZORPAY)

Customer clicks PAY NOW

After payment success:

• Verify payment
• Update status = PAID

-----------------------------------------------------

6️⃣ AUTO CONFIRMATION SYSTEM

After payment:

Automatically send:

A) WhatsApp (BotBee API)
B) SMS (Fast2SMS)
C) Email

Invoice: {link}

Thanks - Al Burhan Tours & Travels"
