Build a full-stack online booking system for a travel company named
"Al Burhan Tours & Travels".

The system must support a booking approval workflow where customers request bookings and the admin approves them before payment.

TECH STACK
Frontend: React / Next.js
Backend: Node.js + Express
Database: MongoDB
Authentication: OTP login using Fast2SMS
Payment Gateway: Razorpay
Messaging: WhatsApp (BotBee API), SMS (Fast2SMS)

------------------------------------------------

BOOKING WORKFLOW

1. Customer opens package page.
2. Customer clicks "Request Booking".
3. Customer enters details:

Name
Mobile Number
Email
Number of Pilgrims
Passport details
Preferred departure date
Package selected

4. System creates booking with status:

PENDING APPROVAL

5. Admin dashboard shows new booking request.

Admin options:
Accept Booking
Reject Booking

------------------------------------------------

IF ADMIN ACCEPTS BOOKING

Booking status changes to:

APPROVED

System sends message to customer:

"Your booking request has been approved. Please complete payment to confirm your booking."

Customer receives notification via:

Email
SMS
WhatsApp
RCS message

------------------------------------------------

PAYMENT PROCESS

Customer logs in to dashboard.

Customer clicks:

"Pay Now"

Payment methods via Razorpay:

UPI
Credit Card
Debit Card
Net Banking

------------------------------------------------

AFTER PAYMENT SUCCESS

Booking status becomes:

CONFIRMED

System automatically:

Generate Invoice PDF
Send confirmation notifications

Customer receives:

Booking Confirmation
Invoice
Payment receipt

via:

Email
SMS
WhatsApp
RCS
